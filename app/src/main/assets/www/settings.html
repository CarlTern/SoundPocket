Voice Control

Toggle this slider to activate voice control.

When having voice control activated you can change sound package by using the following voice commands:

Activate + package name

Ex. to activate the Mario sound package say:

Activate Mario


Hint! The voice control can be a bit tricky and requires you to speak loudly and clear. You may have to practise to get it right.
